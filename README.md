# rocksdb-es

> A RocksDB implementation of the `PersistedEventRepository` trait in cqrs-es.

---

## Usage
Add to your Cargo.toml file:

```toml
[dependencies]
cqrs-es = "0.4.4"
rocksdb-es = "0.1.0"
```
